# Обзор API TaskTracker

TaskTracker API предоставляет RESTful интерфейс для управления задачами, проектами, пользователями и другими сущностями системы.

## Базовый URL

Все API запросы должны отправляться на следующий базовый URL: https://api.tasktracker.example.com/v1

**Пример полного URL:** https://api.tasktracker.example.com/v1/tasks

## Аутентификация

### API ключи

Для доступа к API необходимо использовать API ключ. Получить ключ можно в личном кабинете пользователя.

#### Использование API ключа

Добавьте API ключ в заголовок `Authorization` всех запросов:

```http
Authorization: Bearer ваш_секретный_ключ_здесь
